## Gamig0 (27 Feb 2020)

Java 有 8 种基本数据类型，但在《Thinking of Java》一书中，把 void 也算作一种基本数据类型，包装类 Void。在《码处高效》中，把引用变量(refvar)也算作一种基本数据类型。默认值为 null，存储 refobj 的首地址。
